HTML & CSS 隨手記

October 1, 2020 Aiden
清單標籤：
  • 空心圓：list-style: circle
  • 實心方：list-style: square
  • 實心圓(ul)：list-style: disc
  • 數字(ol)：list-style: decimal
  • 圖片：list-style: list-style-image:url("img/sample.jpg");
文字斷落的設定：
  • 字型：font-family: verdana
  • 行距：line-height: 1.5/24px
  • 字距：letter-spacing: 2px
  • 段落前空格：text-indent: 32px
  • 劃線：text-decoration: underline(底線)/line-through(刪除線)
CSS3技巧
  • div固定寬高：box-sizing: border-box(寬高設多少就多少，不會因padding/border影響)
  • 圓弧效果：border-radius: 10px 0 0 10px(左上順時鐘)
  • 漸層效果：background: linear-gradient(black, yellow)
  • 陰影效果：box-shadow: 5px 5px 15px gray(x軸,y軸,柔焦效果,顏色)
  • 圖片填滿區塊：background-size: cover; background-position: center center;
  • 元素水平居中：position: absolute; left: 50%; transform: translateX(-50%);
  • 背景視差滾動：background-attachment: fixed;
  • 自定義的速度曲线[通常用于 CSS 过渡（transitions）和动画（animations）])：cubic-bezier(1, 0.08, 0, 1.19)
  • CSS動畫：animation和@keyframes
Flex排版技巧
  • flex相關屬性
    1. 必備屬性：display: flex
    2. 軸線：flex-direction
    3. 換行：flex-wrap
  • flex-direction主軸方向
    1. 左→右：row(default)
    2. 左←右：row-reverse
    3. 上↓下：column
    4. 上↑下：column-reverse
  • justify-content主軸對齊
    1. 左→右：flex-start
    2. 左←右：flex-end
    3. 置中：center
    4. 間距均分(不貼框)：space-around
    5. 間距均分(貼框)：space-between
  • align-item交錯軸對齊/align-content交錯軸對齊(有wrap變多列時)/align-self單一子項目的交錯軸對齊
    1. 上↓下：flex-start
    2. 上↑下：flex-end
    3. 置中：center
    4. 滿版：stretch
    5. 行內對齊：baseline
  • align-self其中之一對齊
    1. 上↓下：flex-start
    2. 上↑下：flex-end
    3. 置中：center
    4. 滿版：stretch
    5. 行內對齊：baseline
SCSS/SASS
  • 使用prepros和sublime
  • SCSS：類似CSS但可直接在第一層的{}內新增第二階層的東西
  • SASS：不用括號，所有階層是透過縮排來判斷
  • 可宣告變數：$variables: 參數值 ex. $font-m: 16px;
  • @import－合併別的scss： ex. @import 'reset';
  • 被合併的scss檔名要加_： ex. _reset.scss
  • @mixin－SCSS的function： ex. @mixin circle($size, $bgcolor) { font-size : $size / 3; background : $bgcolor; }
  • @include－使用function： ex. @include circle(30px, #fff)
  • @content－mixin內的內容可自定義
