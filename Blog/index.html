Angular

Angular隨手記

March 19, 2024 Aiden

Angular 是一個廣受歡迎的前端開發框架，用於建構現代且高效的 Web 應用程序。它以模組化架構著稱，讓開發者能夠輕鬆分解應用程序並實現團隊合作和代碼重用。Angular 的雙向數據綁定功能使得管理應用程序狀態變得更加容易，並提供了豐富的功能集，如服務、指令、表單處理和路由，幫助開發者建構高質量、易於維護的 Web 應用。

Git隨手記

October 18, 2023 Aiden

Git 是一種分散式版本控制系統，用於跟蹤代碼變化和協作開發。GitHub 則是基於 Git 的代碼託管平台，讓開發者能夠儲存、分享和協作開發代碼，同時提供了問題跟蹤、Wiki、部署等功能。這些工具為開發者提供了方便和高效的代碼管理和協作開發方式。

Bootstrap

Bootstrap隨手記

October 17, 2022 Aiden

Bootstrap 是一個流行的開源前端框架，用於快速構建具有響應式和現代化設計的網站和 Web 應用程序。它提供了許多可重用的組件和樣式，包括網格系統、排版工具、表單元件和按鈕，同時支持響應式設計，確保在不同尺寸的設備上提供一致且專業的用戶體驗。Bootstrap 為開發者提供了快速構建美觀且具有彈性的網站和應用程序的工具和資源。
