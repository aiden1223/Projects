jQuery隨手記

November 1, 2023 Aiden
jQuery語法：
  • 取消默認事件：e.preventDefault();
  • 隱藏/顯示：hide()/show()
  • 增加html標籤屬性：attr('屬性','屬性值') ex.$('#googlelink').attr('href','https://google.com.tw')
  • 漸出/漸入：slideUp()/slideDown()/slideToggle() 可設定毫秒數
  • 淡出/淡入：fadeOut()/fadeIn()/fadeToggle() 可設定毫秒數
  • 開啟或隱藏：toggle()
  • 增加或移除class：addClass('class名稱')/removeClass('class名稱')/toggleClass('class名稱')
  • 取消默認行為：event.preventDefault()
  • 調整css屬性：css('width','500px')
  • 延遲效果：delay()
  • 中斷效果：stop()
  • 操作元素本身的父元素：$(this).parent().addClass()
  • 操作元素本身的同層元素：$(this).siblings().addClass()
  • 操作元素本身的子層元素：$(this).find('h1').addClass() (需指定子元素標籤)
  • 插入標籤或文本：html('< h1>插入h1標題< /h1>'')/text('插入文本')
  • 第三方插件：動畫效果－animate.css
  • 第三方插件：字型圖片－fontAwesome
  • 淺層/深層複製：jQuery.extend()
  • AJAX：
    1. $.ajax({
        url: 'https://example.com/api/getData', // 請求的 URL
        type: 'GET', // 請求類型
        dataType: 'json', // 預期服務器回應的數據類型
        success: function(data) {
          // 請求成功時的回調函數
          console.log(data);
        },
        error: function(jqXHR, textStatus, errorThrown) {
          // 請求失敗時的回調函數
          console.error('Request failed: ' + textStatus);
        }
      });
    2. $.ajax({
        url: 'https://example.com/api/postData', // 請求的 URL
        type: 'POST', // 請求類型
        dataType: 'json', // 預期服務器回應的數據類型
        data: { // 發送到服務器的數據
          key1: 'value1',
          key2: 'value2'
        },
        success: function(data) {
          // 請求成功時的回調函數
          console.log(data);
        },
        error: function(jqXHR, textStatus, errorThrown) {
          // 請求失敗時的回調函數
          console.error('Request failed: ' + textStatus);
        }
      });
