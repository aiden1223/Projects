November 1, 2022 Aiden
javascript語法：
  • 指定ID：var element = document.getElementById('id'); // 可以將這段設為一個變數方便閱讀
  • 修改內容：element.textContent('');
  • 取得型態：element.type
  • 取得值：element.value
  • 設置/獲取元素內容：element.innerHTML = '...'; / var content = element.innerHTML;
  • 新增節點：var newElement = document.createElement('em'); newElement.textContent = '這是新增的文本';
  • 增加子結點：var parentElement = document.getElementById('parent'); parentElement.appendChild(newElement);
  • function宣告：
  • 確認型態：var x = 5; console.log(typeof x); // 输出 "number"
  • 單一選擇器：document.querySelector('.title a')
  • 修改CSS樣式：document.querySelector('.rocket-1').style.bottom = '2000px';
  • 字串轉數字：parseInt()
  • 多選擇器：document.querySelectorAll('.title a')。(多筆資料會回傳陣列)
  • 增加標籤屬性：.setAttribute('href','http:www.yahoo.com.tw')
  • 取得標籤屬性：.getAttribute('href')
  • 給預設值 -> ||：var data = JSON.parse(localStorage.getItem('listData')) || []
  • 立即函式：(function (){ console.log('立即函式') } ())
  • 可以查看object內建函式：var e = new String('字串');
  • a && b。若a為Truthy(真值)回傳b，若a是Falsy(假值)回傳a
  • a || b。若a為Truthy(真值)回傳a，若a是Falsy(假值)回傳b
  • 了解一个對象包含哪些属性和方法：console.dir()
  • 在將a變數賦值給b變數時，一般型別皆是傳值，僅有物件(object,array,function)是傳參考，意即a,b都參照同個目標，若任一變數進行異動，兩個變數的結果會是一樣的
  • 淺層/深層複製：Object.assign()
  • map()：會將原陣列運算結果回傳回去
  • for (var key in person) { console.log(`列舉${key}`); }

 event事件
  • 點擊事件(只會執行最後一個)：.onclick() = function(e){}
  • 事件監聽：.addEventListener('click',function(e){},false)
      參數：選擇事件,匿名function,false/true
    1. false(事件氣泡Event Bubbling:從指定元素往最外層)
    2. true(事件捕捉Event Capturing:從最外層往內找指定元素)
    3. 終止冒泡事件：e.stopPropagation();
      事件
    1. 表單內容更動內容時觸發：change
    2. 按下鍵盤：keydown
    3. 所在焦點：focus
    4. 離開焦點：blur
    5. 當滑鼠滑入指定內容時觸發：mousemove
  • 取消元素的默認行為：e.preventDefault()
  • 了解目前所在元素位置：e.target/e.target.nodeName
  • 鍵盤代碼：e.keyCode
  • 滑鼠位置(依裝置解析度：screen；依網頁本身：page；依瀏覽器視窗：client)
    1. e.screenX
    2. e.screenY
    3. e.pageX
    4. e.pageY
    5. e.clientX
    6. e.clientY
    7. .mouseImg { position:fixed } el.addEventListener('mousemove', getPosition, false); function getPosition(e){ var mouseImg = document.querySelector('.mouseImg'); mouseImg.style.top = e.clientY + 'px'; mouseImg.style.left = e.clientX + 'px'; }
localStorage-瀏覽器資料儲存
  • localStorage僅能儲存sting資料
  • "儲存"localStorage key值：localStorage.setItem("myName",str)
  • "取得"localStorage key值：localStorage.getItem("myName")
  • 將object轉換成sting：JSON.stingify(object)
  • 取得轉換成sting的object值再轉回object：JSON.parse(localStorage.getItem("myName"))

瀏覽器功能簡介

  • 下一頁：window.history.forward()
  • 開啟列印：window.print()
  • 另開視窗：window.open('http://')

  • 瀏覽器整個高度：window.outerHeight

 AJAX
  • 建立變數：var element = new XMLHttpRequest();
  • 取得狀態：element.readyState
    1. 0: 已產生一個XMLHttpRequest，但還沒連結資料
    2. 1: 使用了open()，但還沒傳送資料
    3. 2: 偵測到有用send
    4. 3: loading
    5. 4: 有撈到資料了，數據已經完全接受到
  • 開啟連結請求：element.open('GET','https',true)
    1. .open變數(格式, 要讀取的網址, 同步非同步)
    2. GET(讀取資料)、POST(傳送資料到伺服器)
    3. 同步非同步：true(非同步，不會等資料回傳，就讓程式繼續跑、false(同步，等資料回傳程式才會繼續跑)
  • 發送請求
    1. GET：element.send()
    2. POST：element.send('email=aiden@gmail.com&password=123456') (要傳遞的參數和參數值)
  • 取消請求：element.abort();
  • 設定回應類型：element.responseType = 'json'; (default:text)
  • 處理回應：element.onload = function() { if (element.status >= 200 && element.status < 300) { // 請求成功，處理響應數據 console.log(element.response); } };
  • HTTP狀態碼：element.status (200至299的狀態碼範圍被認為是「成功」)
  • 偵測是否有開啟CORS：該網域有開啟CORS，才能跨網域撈取資料
  • post前要宣告格式：element.setRequestHeader("Content-type","");
    1. application/x-www-form-urlencoded
    2. application/json
