RWD隨手記

November 1, 2023 Aiden
RWD viewport指令：
  • 使用RWD時<head>內需加上：<meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  • 畫面寬度：device-width
  • 畫面高度：device-height
  • 初始縮放比：initial-scale
  • 最小可縮放比：minimum-scale
  • 最大可縮放比：maximum-scale = 2.0 (最大兩倍)
  • 使用者是否可縮放：user-scalable = no (否)
CSS 3 Media Query：
  • 由大至小，當寬度在768px以下時(即符合最大寬度)title class內文字為藍色：@media(max-width:768px){ .title{color : blue;} }
  • 由小至大，當寬度在768px以上時(即符合最小寬度)title class內文字為藍色：@media(min-width:768px){ .title{color : blue;} }

Float RWD 排版方式：
  • 三欄式流體設計
    1. .wrap { max-width: 1024px; }
    2. .content li { width: 31.33333%; margin: 1% 0.5em; float: left;}
    3. @media (max-width:768px) { .content li {width: 48%} }
    4. @media (max-width:569px) { .content li {width: 98%} }
  • 左側選單固定、右側流體式設計
    1. .wrap { max-width: 1024px; }
    2. .menu { width: 200px; float: left;} (固定寬度)
    3. .content { margin-left: 200px;} (往左推menu的寬度)
  • header橫式menu隱藏變漢堡選單
    1. html需有個隱藏選單
    2. .header { position: relative; }
    3. .menu { float: right; }，.menu li { float: left; }使選單並排
    4. @media .menu { position: absolute; max-height: 0px; overflow: hidden;} 將原本選單設為絕對定位並隱藏
    5. @media .menu { top: 81px; left: 0; right:0; transition: max-height 2.3s;} 將選單定位於header(80px+1px)下方，並使左右滿版，加上漸變特效
    6. @media .menu li{ float: none;} 取消選單的並排
    7. @media .showmenu{ display: block; float: right; } 使漢堡選單變區塊元素且靠右
    8. js $('.showmenu').on('click', function(e) { $('body').toggleClass('menu-show'); }); 點擊漢堡選單後在body加上menu-show的class
    9. @media .menu-show .menu { max-height: 500px } 賦予選單高度即會出現
RWD 注意事項：
  • 不能有X軸bar：width改為max-width
  • 善用百分比(xx.xxxxx%)：依據父元素max-width的百分比去自適應寬度
  • 中文內文單行字元 30～40 會比較好閱讀，英文則是 32～80 個字元數
  • 手指點選範圍高度約在44px
  • 善用touch點擊事件
  • Viewport Sizes 服務網站
  • iPad:768px、iPhone:767px以下
  • 使某div區塊(表格)產生X bar：overflow-x: auto;
  • input欄位小技巧，使用type="email","tel","number"，來設定手機鍵盤預設輸入方式
  • 圖片顯示：max-width及width:100%，差異在width會隨著父元素去延展，而max-midth會保持原圖的大小
  • CSS reset建議增加：img{ max-width:100%; height: auto; }
  • 若有使用到圖片的話，盡量選擇大一點的圖片(600px)，避免自適應時因為圖片太小放大糊掉的問題
  • TinyPNG壓縮圖片
  • RWD最小寬度一定要兼容到320px，因為是全世界最小手機尺寸(iphone5)，大約20個字
