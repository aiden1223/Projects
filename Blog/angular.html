Angular隨手記

November 1, 2023 Aiden
  • Angular專案建置：
    1. 新增專案資料夾 ex. demoAngular
    2. 開啟CMD後輸入ng new demoAngular (資料夾名稱)
    3. 輸入cd demoAnuglar進入資料夾後，輸入npm start 就會將專案Run起來 (Ctrl + C-中斷)
    4. 輸入code . 即可開啟VSCode
  • Component
    1. 新增Component：ng generate component page1(名稱) (縮寫：ng g c page1)
    2. 新增後要手動在app.component.ts內加入：

      才可以在app.component.html內加上< app-page1>< /app-page1>
  • 資料繫結
    1. 內嵌繫結：
      • 用兩個{{}}將參數括起來：{{property}}
        • 									
HTML：
< h1>hello, {{ myname }}

TS：
myname = 'Aiden';
    2. 屬性繫結：
      • 將HTML標籤內屬性用[]括起來後給予參數：[property]=’statement’
        • 									
HTML：
< a [href]="url">Google首頁

TS：
url = 'https://www.google.com/';
      • 若為HTML5 data-*屬性，則要在前面加上attr.
        • 									
HTML：
< div id="example" [attr.data-name]="myname">

TS：
myname = '這是屬性繫結';
    3. 事件繫結 (顯示所有可用的方法：Ctrl + 空白)
      • 將事件用()括起來後給予function：(event)='someMethod($event)'
        • 									
HTML：
< p (click)="changeText()">點擊後變更myname

TS：
changeText(){
	this.myname = '(事件繫結)';
};
      • 使用DOM的Event，利用DOM物件的Poperty進行判斷：
        • 									
HTML：
< p (click)="changeText($event)">點擊後變更myname

TS：
changeText($event: any){
	console.log($event); // 確認是什麼Event (MouseEvent)
	// 將any改為MouseEvent後，就可以享用到intellisense的功能，輸入 . 後便會自動帶出可用功能
	if ($event.altKey) this.myname = '按下Alt後改變(事件繫結)';
};
    4. 雙向繫結 (屬性和事件)
      • [(ngModel)]='property'
        • 									
HTML：
< input type="text" placeholder="請任意輸入" [(ngModel)]="word">< br>
< p>您輸入的字為"{{word}}" 共有{{word.length}}個字

TS：
import { FormsModule } from '@angular/forms'; // 引入 FormsModule

@Component({
	...
	imports: [RouterOutlet, Page1Component, FormsModule], // 添加FormsModule
	...
  })

  export class AppComponent {
	...
	word = ''; // 先宣告word參數
	// input輸入後便雙向繫結回TS，再透過內嵌繫結回HTML上顯示
  }
  • 範本參考變數 (在HTML自訂變數名稱ex. tKeyword)
    1. 基本操作方式
      2. 							
HTML：
< input type="text" placeholder="請任意輸入" #tKeyword>< br>
< p>您輸入的字為"{{tKeyword.value}}" 共有{{tKeyword.value.length}}個字
    2. 延伸應用: 可將整個A Component設為範本變數，至B Component對A Component進行操作
      3. 							
HTML：
< app-header #tHeader>
< p (click)="tHeader.title=tKeyword.value">Header:您輸入的字為"{{tKeyword.value}}" 共有{{tKeyword.value.length}}個字
  • 構造函數constructor (主要用於初始化工作)
      • 							
TS：
constructor() {
    setTimeout(() => {
      this.title = '兩秒後顯示內嵌繫結'
    }, 2000);
  };
  • 資料繫結
    1. 元件型指令
      • 預設「元件」就是一個含有樣板的指令(最常見)
        •
    2. 屬性型指令
      • 這種指令會修改元素外的外觀或行為
      • 例如內建的 NgStyle 或 NgClass 指令就可讓你自由變更樣式
        • －NgStyle
									
TS：
import { CommonModule } from '@angular/common';
@Component({
	．．．
	imports: [RouterOutlet, Page1Component, FormsModule, HeaderComponent, CommonModule], // 添加CommonModule
	...
})

counter = 0;

changeTitle($event: MouseEvent) {
    ...
    this.counter++;
}

getStyle() {
	return { 'font-size': (12 + this.counter) + 'px' }
}

HTML：
< p (click)="changeTitle($event)" [ngStyle]="getStyle()">點此換標題且放大字體，共點擊了{{counter}}次

// 或是在style.加上想要動態的樣式ex.font-size
< p [style.font-size]="(12+counter)+'px'">動態樣式顯示{{counter}}
									
								
        －NgClass：[ngClass] = "{ClassName : bool}"
									
HTML：
// 透過布林值判斷要不要套用這個Class
< p [ngClass]="{'hiehlight': counter% 2==2}">ngClass
    3. 結構型指令 (Structure Directives)
      • 這種指令會透過新增和刪除DOM元素來改變DOM結構
      • 例如內建的 Ngif / NgFor / NgSwitch 就可以用來控制DOM結構
        •
  • Angular相關指令
    1. 查看所有可新增的元件：ng generate -h
    2. 建置專案：ng build (會產生dist資料夾，未來需丟到IIS內)(ng build -prod，會將js壓縮)
    3. 自動換行：Alt + Z
    4. 確認版本：ng -v
    5. 更新Angular版本：ng update
    6. 確認Angular CLI版本：
      npm list -g -depth=0
      npm outdated -g (沒東西就是最新版本)
    7. 安裝Angular CLI：npm install -g @angular/cli
  • Note
    1. F1 -> reload，重新載入視窗
    2. Encapsulation: ViewEncapsulation.None // 不封裝A Component的CSS樣式，渲染到全專案，加在A的ts的@Component內
